# LambdaONT
FASTQ and reconstructed genome of Lambda phague. ONT Essay. 

In this repository, you'll find, fastq file that contains basecalled data with ONT Albacore Sequencing Pipeline Software (version 2.1.10) from Lambda Phague and also the reconstructed genome by minimap2/Miniasm polished with Racon. It was sequenced with 9.4 chemistry and SKQ-RAD002 kit. This data was obtained by: Javier Correa, JF Díaz-Nieto, Diana Rincón-Toquica and Nicolás David Franco Sierra. Analyzed by the owner of the repository. In the Universidad EAFIT.
